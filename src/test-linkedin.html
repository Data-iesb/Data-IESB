How to Add/Manage LinkedIn Profiles

Using the Management Script:

# List all team members and their LinkedIn status
./manage-linkedin.sh list

# Add LinkedIn profile to a team member
./manage-linkedin.sh add 3 https://www.linkedin.com/in/natalia-evangelista

# Remove LinkedIn profile from a team member  
./manage-linkedin.sh remove 5

# Show only members with LinkedIn profiles
./manage-linkedin.sh show-with-linkedin

Using AWS CLI directly:

# Add LinkedIn profile
aws dynamodb update-item \
  --table-name DataIESB-Team \
  --key '{"id":{"S":"3"}}' \
  --update-expression "SET linkedin = :linkedin, updatedAt = :updatedAt" \
  --expression-attribute-values '{
    ":linkedin":{"S":"https://www.linkedin.com/in/natalia-evangelista"},
    ":updatedAt":{"S":"2025-08-10T15:18:00.000Z"}
  }'
